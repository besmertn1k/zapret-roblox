2. Распаковка
Создайте новую папку на компьютере (например, Zapret)

Распакуйте всё содержимое архива в эту папку

3. Обновление файлов
Замените содержимое папки ipset-all на файл из репозитория (ipset all)

Также обновите содержимое папки list-general

4. Запуск и настройка
Запустите файл service.bat от имени администратора

В открывшемся меню убедитесь, что:

Пункт 6 имеет значение enabled

Пункт 7 измените на enabled

Пункт 8 установите на any

Нажмите цифру 9 для применения настроек

После этого пункт 8 должен измениться на loaded

5. Выбор оптимального конфига (general)
В меню нажмите цифру 11

В новом окне выберите:

Сначала пункт 1

Затем снова пункт 1

Начнётся сканирование доступных конфигов (примерно 4 минуты)

Дождитесь завершения сканирования

В результате программа покажет Best config — оптимальный для вас вариант (например, ALT11)

Закройте окно и вернитесь в главное меню

6. Запуск выбранного конфига
В главном меню выберите номер, соответствующий вашему лучшему конфигу (например, для ALT11 нажмите 11)

Дождитесь сообщения об успешном применении

7. Проверка работы
Откройте Discord или YouTube и проверьте доступ

Если что-то не работает:

Перезапустите программу от администратора

Убедитесь, что в системе нет конфликтующих VPN-сервисов

Попробуйте другой конфиг из списка

Важно!
Программа должна оставаться запущенной для работы обхода блокировок

При обновлении списков IP или версии программы повторяйте шаги 4–7

Примечание: Данная инструкция актуальна для версии Zapret Discord YouTube 1.9.2. Для других версий шаги могут отличаться.
