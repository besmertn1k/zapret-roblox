Полное руководство по установке и настройке Zapret (v1.9.2) для разблокировки Roblox.

Перед началом: убедитесь, что у вас есть права администратора на компьютере. Инструкция актуальна для версии 1.9.2.

Шаг 1: Скачивание и распаковка
1. Скачайте архив программы по официальной ссылке: https://github.com/Flowseal/zapret-discord-youtube/releases/download/1.9.2/zapret-discord-youtube-1.9.2.rar
2. Создайте на компьютере отдельную папку (например, C:\Zapret).
3. Распакуйте ВСЁ содержимое скачанного архива в созданную папку.

Шаг 2: Обновление файлов (важно!)
Перед первым запуском необходимо обновить списки IP-адресов:
1. Откройте распакованную папку. Найдите папку "ipset-all".
2. Удалите всё её содержимое.
3. Скачайте новый файл "ipset-all" из репозитория разработчика (ищите там же, где скачивали архив) и поместите его в пустую папку "ipset-all".
4. Проделайте те же действия (очистка + замена) для папки "list-general".

Шаг 3: Первый запуск и базовая настройка:
1. В папке с программой найдите файл "service.bat". Щёлкните по нему правой кнопкой мыши и выберите "Запуск от имени администратора".
2. Откроется текстовое меню. 
Вам нужно проверить и изменить три пункта:
   - Пункт 6: должен быть в состоянии "enabled". Если нет, нажмите клавишу 6 и Enter.
   - Пункт 7: измените на "enabled". Нажмите клавишу 7 и Enter.
   - Пункт 8: установите значение "any". Нажмите клавишу 8, Enter и выберите "any" из списка.
3. После настройки нажмите клавишу 9 и Enter, чтобы применить изменения.
4. Успешная настройка подтвердится, когда напротив пункта 8 появится статус "loaded".

Шаг 4: Автоподбор лучшего сервера (конфига)
1. В главном меню нажмите клавишу 11 и Enter. Откроется новое окно.
2. В этом окне: сначала нажмите 1 (Start scan) и Enter, затем снова нажмите 1 и Enter для запуска полного сканирования.
3. Сканирование займёт около 4 минут. Дождитесь его окончания.
4. В результате вверху окна будет показан "Best config" — оптимальный для вас вариант (например, ALT11). ЗАПОМНИТЕ ЕГО НОМЕР или название.
5. Закройте окно сканирования (обычно клавишей 0), чтобы вернуться в главное меню.

Шаг 5: Активация выбранного конфига
1. В главном меню нажмите цифру, которая соответствует вашему лучшему конфигу. Например, если был найден ALT11, нажимаем 11. Если ALT5, нажимаем 5.
2. Дождитесь сообщения об успешном применении конфигурации.

Шаг 6: Проверка работы
1. Откройте браузер и попробуйте зайти на YouTube или запустите Discord. Доступ должен появиться.
2. Окно программы должно оставаться открытым (его можно свернуть).

Если не работает:
- Перезапустите "service.bat" от имени администратора и повторите шаги 3-5.
- Временно отключите антивирус, брандмауэр, другие VPN-сервисы и прокси.
- Попробуйте вручную выбрать другой конфиг из списка (например, ALT3 или ALT5).

Важные напоминания:
1. Программа должна быть запущена, чтобы обход блокировок работал.
2. При обновлении списков IP-адресов или самой программы необходимо повторить шаги с 3 по 5.
3. При смене сети (например, переходе на другую Wi-Fi сеть) рекомендуется повторно выполнить Шаг 4 (автоподбор конфига).
4. Данная инструкция написана для версии 1.9.2. Для других версий шаги могут отличаться.
